# Network Topologies using Cisco Packet Tracer 🌐

## 📌 Project Overview

This project demonstrates different types of network topologies using Cisco Packet Tracer. Each topology is designed and simulated to understand how devices communicate in various network structures.

---

## 🧰 Tool Used

* Cisco Packet Tracer

---

## 🏗️ Topologies Implemented

### ⭐ 1. Star Topology

* All devices are connected to a central switch
* Easy to manage and troubleshoot

### ⭐ 2. Bus Topology

* Devices are connected along a single communication line
* Simple but less reliable

### ⭐ 3. Ring Topology

* Devices are connected in a circular structure
* Data travels in one direction

### ⭐ 4. Mesh Topology

* Every device is connected to multiple devices
* High reliability and redundancy

### ⭐ 5. Tree Topology

* Combination of star and bus topology
* Used in hierarchical networks
